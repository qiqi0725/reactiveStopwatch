## [Stopwatch Demo](https://stopwatch-reactive.herokuapp.com/) | [Documentation](https://qiqi0725.github.io/reactiveStopwatch/index.html)

**To Run Locally**
* [Install npm with Node](https://nodejs.org/en/download/)
* Update local repo
```sh
npm update
```
* Run start script
```sh
npm run start
```
