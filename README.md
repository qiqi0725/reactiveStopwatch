## A stopwatch created with React
## [Stopwatch Demo](https://www.google.com) | [Documentation](https://vboivin.github.io/geolocator/index.html)

**To Run Locally**
* [Install npm with Node](https://nodejs.org/en/download/)
* Update local repo
```sh
npm update
```
* Run start script
```sh
npm run start
```
